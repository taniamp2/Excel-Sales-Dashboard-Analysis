# Excel Sales Dashboard

This project presents a simple sales performance dashboard built in Microsoft Excel using pivot tables and charts.

## Objectives

- Analyze total revenue
- Compare revenue by region
- Compare revenue by product category
- Create visual dashboard for quick business insights

## Dashboard Features

- Total revenue KPI
- Revenue by region visualization
- Revenue by category visualization
- Pivot table analysis

## Key Insights

- Total revenue: 4780
- North America generates the highest revenue
- Electronics dominate total sales
- Office supplies contribute the least revenue

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
